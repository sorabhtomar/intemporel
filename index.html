Intemporel

Intemporel is a responsive WordPress theme. It's a modern theme with clean layout and standard typography inspired from Google's material design. Intemporel is a responsive WordPress theme for blogs and magazines. It's based on _s framework by Automattic.

Features

Colors: Theme colors are inspired by Google's material design with primary color being Indigo and secondary color as Pink. Users can cutsomize colors in Theme Customizer. You can choose any color you would like or choose from material design color palettes from Google.

Fonts: Intemporel uses Roboto font designed by Christian Robertson. The font was updated recently when Christian Robertson and the Material Design team unveiled the latest version of Roboto at Google I/O last year. It's available under Apache License, version 2.0.

Asides: Intemporel supports 2 sidebars. The theme has a single sidebar for archive pages, and another content sidebar on Single Post pages & Page post type.

Icons: Intemporel uses Automattic's Genericons icon font.

Sliding Navigation: The main site navigation can be opened by a menu button. The navigation slides from the top (from behind the header). On mobile devices, it slides from left. The navigation is 3-level deep.

Site header animation: The site header in Intemporel is 100px in height. On scroll down, it reduces itself to 48px and is fixed on the top, giving more room to your content while providing quick access to menu and search buttons.

Featured Posts: Intemporel uses a special template for the first post loaded on the homepage and archive pages.

Subtle Shadows: The theme comes with subtle shadows around boxes contributing in the overall beauty of Intemporel theme.

Installation

The theme is in development stage and many more changes are in the way. To install the theme, like other WordPress themes:

  • Download theme from WordPress repository.
  • Extract the folder intemporel in wp-content/themes directory of your WordPress installation.
  • Activate it from Admin Dashboard.
